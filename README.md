# SAC-Plus 

Sistema web para la gestión de monitoreo de equipos, desarrollado con Spring Boot, MySQL y tecnologías web.

---

##  Funcionalidades

- Inicio de sesión con validación en base de datos
- Registro de monitoreo
- Listado de registros
- Búsqueda por nombre
- Actualización de datos
- Cambio de estado (Finalizado)

---

##  Tecnologías utilizadas

- Java
- Spring Boot
- Spring Data JPA
- MySQL
- HTML, CSS, JavaScript

---

##  Configuración

1. Crear la base de datos en MySQL
2. Configurar credenciales en:
   src/main/resources/application.properties
3. Ejecutar el proyecto desde:
   SacplusApplication.java
4. Usuario de prueba:

   -Usuario: admin

   -Contraseña: admin
